КОНКРЕТНЫЕ ДЕЙСТВИЯ ПО ЗАГРУЗКЕ

1. Сделайте резервную копию текущей папки kreditor-site.
2. Распакуйте архив kreditor-site-final-v5.zip.
3. Откройте локальную папку kreditor-site, связанную с GitHub.
4. Удалите старые видимые файлы из kreditor-site. Скрытую папку .git НЕ удаляйте.
5. Скопируйте внутрь kreditor-site всё содержимое папки kreditor-site-final-v5.
6. Откройте kreditor-site в VS Code.
7. Terminal → New Terminal.
8. Выполните по очереди:
   git add .
   git commit -m "KREDITOR V5 final site"
   git pull origin main --rebase
   git push origin main
9. Откройте GitHub → Actions и дождитесь зелёной галочки.
10. Откройте https://kreditor.pro в приватном окне Safari.
11. Проверьте главную, все пункты меню, формы и мобильную версию.

ОТКАТ:
git log --oneline
git revert <номер_предыдущего_коммита>
git push origin main
