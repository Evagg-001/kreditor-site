# KREDITOR V8 Final — Changelog

## Пользовательские улучшения
- В модальном окне добавлены быстрые способы связи:
  - телефон;
  - WhatsApp;
  - Telegram;
  - email.
- Сохранён основной сценарий отправки формы через WhatsApp.
- Контактные действия адаптированы для мобильных устройств.
- Для контактных ссылок добавлены понятные подписи и доступные `aria`-атрибуты.

## Технические изменения
- Build ID обновлён до `KREDITOR-V8-20260712-FINAL`.
- Добавлено событие аналитики `modal_contact_click`.
- Сохранены события звонка, WhatsApp, Telegram, email и формы.
- Проведена проверка локальных ссылок и ресурсов.
