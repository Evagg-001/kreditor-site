# KREDITOR Master Release 5.0 — Архитектура

## Назначение
Статический корпоративный сайт для GitHub Pages с единым визуальным стилем, внутренними страницами, SEO-файлами и формами связи.

## Основные части
- `index.html` — главная страница.
- `about.html`, `practices.html`, `academy.html`, `partners.html`, `faq.html`, `contacts.html` — внутренние страницы.
- `practice-*.html` — страницы юридических практик.
- `legal/` — правовые документы.
- `assets/` — изображения и favicon.
- `styles.css` — единая дизайн-система.
- `script.js` — навигация, формы, модальные окна, cookie-баннер.
- `robots.txt`, `sitemap.xml`, `CNAME` — публикация и SEO.

## Принципы
- Один набор стилей и скриптов для всех страниц.
- Публичный сайт остаётся статическим и быстрым.
- Новая функциональность подключается отдельными релизами через ветку `develop`.
