# Краткое руководство администратора

- Тексты страниц редактируются в соответствующих HTML-файлах.
- Общие цвета, размеры и карточки находятся в `styles.css`.
- Общая интерактивность находится в `script.js`.
- Контакты следует менять одновременно во всех HTML-файлах и в `script.js`.
- Перед публикацией работайте в ветке `develop` и используйте Pull Request в `main`.
