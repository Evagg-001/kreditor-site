# KREDITOR Protection Standard — примечание к тестовой сборке

- Build ID: `KREDITOR-V3-20260710-ALPHA`
- Тестовая версия содержит метатег `noindex`.
- Публичный HTML/CSS/JS нельзя сделать абсолютно недоступным для копирования: браузер должен получить его для отображения.
